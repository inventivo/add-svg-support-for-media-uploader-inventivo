Add SVG Support for Media Uploader

== Description ==

WordPress does not allow to upload .svg files out of the box. This plugin adds SVG support to the media uploader. 

Visit our <a href="https://www.inventivo.de/seo-muenster">Website</a> in order to see the plugin in action.

Inventivo WordPress plugins are small and simple:
- Responsive - fits to all screens
- Easy configuration
- Lighweight code

Get your Add SVG Support for Media Uploader Plugin now!

How it works:

1. Upload the Plugin
2. Activate the Plugin

== Installation ==

Use WordPress' Add New Plugin feature, searching "Add SVG Support for Media Uploader| inventivo", or download the archive and:

1. Unzip the archive on your computer  
2. Upload `inventivo-addsvgsupport` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
